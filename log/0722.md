- 실전프로젝트 2 전체 RAG 파이프라인 및 우선순위 확정
- 임베딩 모델 후보 비교 완료
  - BGE-m3
  - BGE-m3-ko
  - Qwen3-Embedding-8B
  - Nemotron-3-Embed-8B
- 검색 성능과 운영 가능성을 고려하여 **BGE-m3-ko를 최종 임베딩 모델로 선정**
- Dense, BM25, Hybrid 검색 성능 비교
- 질문 유형별 검색 결과를 기반으로 검색기 라우팅 방식 구현
  - 기본 질문: Hybrid 검색
  - `table_lookup`: Dense 검색
- 질문 유형 분류기 구현 및 성능 검증
- Qdrant Local 벡터 저장소 구축
  - 494개 청크 및 1024차원 임베딩 적재
  - 원문, 출처 URL, 페이지 제목, 업무 분류 메타데이터 저장
- Qdrant 데이터 무결성 검증
  - 청크 중복·누락 없음
  - 필수 Payload 필드 누락 없음
  - 원본 임베딩과 저장 벡터 일치 확인
  - 프로세스 재실행 후 데이터 영속성 확인
- 기존 NumPy Dense 검색과 Qdrant 검색의 MRR 및 Top-K 결과가 동일함을 확인
- 평가셋의 정답 문서와 원문 근거 검수 및 오류 문항 수정
- HyperCLOVA X, LangChain·LangGraph 적용 방향 검토
