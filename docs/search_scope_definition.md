# 업무별 검색 범위 정의서

예금보험공사(KDIC) RAG 챗봇의 검색 대상 범위를 업무(도메인) 단위로 정의한다. 챗봇은 사용자 질의를 아래 6개 업무 중 하나로 라우팅한 뒤, 해당 업무에 속한 페이지 안에서 근거를 검색한다. 각 페이지의 출처 URL과 요약(summary)은 `data/corpus.jsonl`에서 추출했다.

- **총 업무 수**: 6개
- **총 페이지 수**: 58개
- **출처**: `data/corpus.jsonl` (page_id / sub_category / source_url / summary)
- **브레드크럼**: 사이트 메뉴 계층 경로(`sub_category`). 검색 라우팅의 하위 범주 근거로 쓴다.
